Adds a recipe to craft any type of Wool into String. The recipe is added to the player's recipe book once they any type of Wool.

```json
{
    "type": "minecraft:crafting_shapeless",
    "ingredients": [
        {
            "tag": "minecraft:wool"
        }
    ],
    "result": {
        "count": 4,
        "item": "minecraft:string"
    }
}
```
